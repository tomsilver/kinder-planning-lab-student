# KinDER Planning Lab

A hands-on lab that builds up to **bilevel planning** on the
[KinDER](https://prpl-group.com/kinder-site/) robotics benchmark: object-centric
states, predicates and classifiers, operators, parameterized options, and a full
SeSamE planner — ending with an open-ended capstone where you write the entire
planning domain for a brand-new pyramid-building environment.

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/tomsilver/kinder-planning-lab-student/blob/main/kinder_planning_lab.ipynb)

Click the badge to open the notebook in Google Colab, then **File → Save a copy in
Drive** to get your own editable copy. The exercises are marked with
`# YOUR CODE HERE`.
